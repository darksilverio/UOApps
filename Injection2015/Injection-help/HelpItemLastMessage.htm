Описание: LastMessage •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.LastMessage()

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Journal, Scripting, Function

Возвращает последнюю строку журнала Injection. Journal(0) возвращает то же что и LastMessage. Предпочтительно использовать функцию InJournal для проверок в журнале.
