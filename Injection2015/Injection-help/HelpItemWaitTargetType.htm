Описание: WaitTargetType •

Синтакс:
Консоль: _WaitTargetType type [color]
Из скрипта: void UO.WaitTargetType(type, [color])

Введено с версии: <=1501.17

Тема: Прицелы
Ключевые слова: Command, Scripting, Targeting, TypeHandling, WaitSystem, Void

Ищет на бекпаке и подсумках обьект по типу (и цвету если задано) после чего устанавливает на этот обьект ловушку прицела.
Следующий прицел, который будет запрошен сервером, будет направлен на этот обьект.
См. так же WaitTargetGround, WaitTargetObject, FindType.
