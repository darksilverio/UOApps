Описание: GetDistance •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetDistance(serial or x1, y1, [x2, y2])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - расстояние до обьекта, которое вычисляется как минимальное из расстояний по осям X и Y.
