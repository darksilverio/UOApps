Описание: ActivateHandle •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: void UO.ActivateHandle()

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, Utility, Void

Пытается вывести на 1 план окно с УО (не работает при свернутом окне).
void uo>ActivateHandle()

Вызов из скрипта:
Code:

sub OnNewPlayer() #Вызывается эвентом при появлении на экране нового игрока
   if uo.GetNotoriety(newplayer) >= 4 then #Гильдвар или ПК
      uo.ActivateHandle()
   endif
end sub
