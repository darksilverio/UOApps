Описание: GetNotoriety •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetNotoriety([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - злобность (цвет):

  • 1 : Innocent (синий)
  • 2 : Friend (зелёный)
  • 3 : Gray (серый)
  • 4 : Criminal (серый)
  • 5 : Enemy (рыжий)
  • 6 : Murderer (красный)
