Описание: SetReceivingContainer •

Синтакс:
Консоль: _SetReceivingContainer [serial]
Из скрипта: void UO.SetReceivingContainer([serial])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, System, Void

Устанавливает приёмную сумку для команд EmptyContainer, Grab.
Установить контейнер по умолчанию (бекпак) можно командой UnsetReceivingContainer.
