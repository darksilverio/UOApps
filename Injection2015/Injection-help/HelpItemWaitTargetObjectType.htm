Описание: WaitTargetObjectType •

Синтакс:
Консоль: _WaitTargetObjectType serial type [color]
Из скрипта: void UO.WaitTargetObjectType(serial, type, [color])

Введено с версии: <=1501.17

Тема: Прицелы
Ключевые слова: Command, ObjHandling, Scripting, Targeting, TypeHandling, WaitSystem, Void

Ищет на земле обьект по типу (и цвету если задано). Устанавливает на указаный обьект ловушку прицела.
Следующий прицел, который будет запрошен сервером, будет направлен на этот обьект.
После срабатывания ловушки будет установлена ловушка на обьект найденый по типу (и цвету если было задано).
См. так же WaitTargetType, WaitTargetObject, FindType.
