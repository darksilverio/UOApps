Описание: CloseHandle •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: void UO.CloseHandle()

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, System, Utility, Void

Закрыть окно клиента Ultima Online.
Работает аналогично uo.CloseUO() но игнорирует дисконнекты.
void uo.CloseHandle()

Вызов из скрипта:
Code:

sub CheckConnection() #Проверяет соединение с сервером
   if not uo.IsOnline() then #Если нет соединения - закрываем клиент (uo.CloseUO() ни к чему не приведет в данном случае)
      uo.CloseHandle()
   endif
end sub
