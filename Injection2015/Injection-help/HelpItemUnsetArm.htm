Описание: UnsetArm •

Синтакс:
Консоль: _UnsetArm name
Из скрипта: void UO.UnsetArm(name)

Введено с версии: <=1501.17

Тема: Одеяние
Ключевые слова: Command, Equipment, Scripting, System, Void

Убирает из памяти запомненый набор вооружения Name. Рекомендуется сделать перед новым запоминанием набора, или включить опцию Unset before set которая делает это автоматически.
