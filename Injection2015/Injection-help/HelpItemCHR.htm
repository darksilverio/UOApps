Описание: CHR •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function CHR(index)

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, Utility, Function

Необходима для отправки сообщения для нажатия Esc в клиенте (другого применения я не нашел)
String CHR(Number index)

Вызов из скрипта:
Code:

sub CancelClientTarget()
   #Очистим консоль от текста (в результате выполнения uo.Say текст из консоли может отправиться на сервер)
   uo.ConsolePrint('')
   uo.Say(CHR(27)) #Эмуляция нажатия Esc в окне УО
end sub
