Описание: ShowJournal •

Синтакс:
Консоль: _ShowJournal [lines_count]
Из скрипта: void UO.ShowJournal([lines_count])

Введено с версии: <=1501.17

Тема: Сообщения
Ключевые слова: Command, Journal, Scripting, System, Utility, Void

Отображает в текстовом окне последние lines_num строк настоящего журнала Injection (по умолчанию 10).
