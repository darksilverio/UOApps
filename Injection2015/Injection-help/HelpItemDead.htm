Описание: Dead •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.Dead()

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Scripting, Utility, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - дохлость. Правильно работает только со своим персонажем, с другими не гарантируется.
