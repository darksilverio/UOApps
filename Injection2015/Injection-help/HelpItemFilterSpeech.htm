Описание: FilterSpeech •

Синтакс:
Консоль: _FilterSpeech command [state]
Из скрипта: void UO.FilterSpeech(command, [state])

Введено с версии: <=1501.17

Тема: Системные
Ключевые слова: Command, Scripting, System, Void

Фильтр речи удалят сообщения подходящие по шаблону.

  • FilterSpeech on - включает фильтр
  • FilterSpeech off - отключает фильтр
  • FilterSpeech clear - очищает фильтр
  • FilterSpeech info - выводит текущий фильтр
  • FilterSpeech add phrase - добавляет шаблон к фильтру. Например если вы добавите фразу What то при включеном фильтре все выражения в которых есть What не будут передаваться клиенту.
  • FilterSpeech remove phrase - удаляет шаблон из фильтра.
