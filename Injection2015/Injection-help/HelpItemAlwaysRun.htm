Описание: AlwaysRun •

Синтакс:
Консоль: _AlwaysRun [state]
Из скрипта: Function UO.AlwaysRun([state])

Введено с версии: <=1501.17

Тема: Приложения
Ключевые слова: Command, Scripting, Utility, Function

Изменяет состояние галочки в опциях клиента Always Run (если клиент поддерживает эту опцию).
При вызове из скрипта:

  • Number uo.AlwaysRun() - возвращает текущее состояние
  • void uo.AlwaysRun(Number state) задает значение для переключения

Вызов из скрипта:
Code:

sub ChangeSpeed()
   uo.AlwaysRun(not uo.AlwaysRun()) #Переключит состояние
end sub


Вызов из хоткеев:
Code:

alwaysrun - переключает состояние галочки
alwaysrun value - устанавливает состояние галочки в value (0 - выключена; иное значение - включена)
