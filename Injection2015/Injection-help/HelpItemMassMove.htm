Описание: MassMove •

Синтакс:
Консоль: _MassMove delay [maxitems color]
Из скрипта: void UO.MassMove(delay, [maxitems, color])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, Utility, Void

Позволяет продублировать перемещение одного эталонного предмета со многими того же типа. Используется для быстрого перемещения и сортировки предметов. Для перемещения ищутся предметы того же типа что и эталонный в том же контейнере, и ложатся в точно то же место куда был положен эталонный. Задержка между перемещениями delay мс (1000 мс = 1 сек). Можно установить предел количества переброщеных предметов maxitems. Если установлен параметр Color то при перебрасывании будет учтён так же цвет эталона.

Для использования сперва введите команду MassMove с параметрами, потом переместите один эталонный предмет вручную.

Для шардов в которых цена предмета задаётся в момент когда предмет ложится в сумку вендору (например AoP, Wolfpack), можно использовать команду MassPriceMove.
