Описание: AddRecall •

Синтакс:
Консоль: _AddRecall book_serial
Из скрипта: void UO.AddRecall(book_serial)

Введено с версии: <=1501.17

Тема: Рунбуки
Ключевые слова: Command, GumpControl, Scripting, Void

Команда служит для добавления заряда рекола в стандартный рунбук Object.
У вас должны быть с собой свитки рекола Type=0x1f4c перед использованием команды.
