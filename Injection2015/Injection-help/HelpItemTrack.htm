Описание: Track •

Синтакс:
Консоль: _Track [state/serial x y]
Из скрипта: void UO.Track([state/serial, x, y])

Введено с версии: <=1501.17

Тема: Приложения
Ключевые слова: Command, Scripting, Utility, Void

Включает указующую стрелку трекинга (Quest Arrow) в клиенте в указаные координаты. 0 - выключить, 1 - включить, Object - включить на положение указаного обьекта. Если задано 1, то X,Y - мировые координаты, куда должна показывать стрелка.
