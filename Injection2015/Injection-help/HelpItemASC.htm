Описание: ASC •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function ASC(text, default)

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, Utility, Function

Конвертирует строку text в число, при невозможности конвертирования указанного текста - возвращает значение default
Number ASC(String text, Number default)

Вызов из скрипта:
Code:

sub IncGlobalValue(ident) #Увеличивает значение указанной глобальной переменной на 1
   var temp = uo.GetGlobal(ident) #Получим значение из глобальной переменной (не инициализированное может вернуть N/A)
   var value = ASC(temp, 0) + 1 #Используем ASC на значение из глобальной переменной и прибавляем 1 к результату
   uo.SetGlobal(ident, str(value)) #Записываем значение обратно в глобальную переменную
   return value #Возвращаем результат Number (если необходимо использовать его в вызывающей функции)
end sub
