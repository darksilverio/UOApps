Описание: DeleteJournal •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: void UO.DeleteJournal([pattern])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Journal, Scripting, Utility, Void

Очищает журнал Injection. Учтите что журнал един для всех скриптов запущеных в данном экземпляре инжекта/клиента. Команда выполняется чтобы отсечь уже отработаные варианты для команды InJournal
