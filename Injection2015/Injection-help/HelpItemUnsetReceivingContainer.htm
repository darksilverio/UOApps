Описание: UnsetReceivingContainer •

Синтакс:
Консоль: _UnsetReceivingContainer
Из скрипта: void UO.UnsetReceivingContainer()

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, System, Void

Устанавливает приёмную сумку по умолчанию (бекпак) для команд EmptyContainer, Grab.
Установить другой контейнер можно командой SetReceivingContainer.
