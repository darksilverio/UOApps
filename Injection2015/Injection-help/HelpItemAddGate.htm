Описание: AddGate •

Синтакс:
Консоль: _AddGate book_serial
Из скрипта: void UO.AddGate(book_serial)

Введено с версии: <=1501.17

Тема: Рунбуки
Ключевые слова: Command, GumpControl, Scripting, Void

Команда служит для добавления заряда гейта в стандартный рунбук Object.
У вас должны быть с собой свитки гейта Type=0x1f60 перед использованием команды.
