Описание: GetSerial •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetSerial([identifier])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - сериал. Команда используется в основном для сравнения системных указателей с другими обьектами, например

Code:
if UO.GetSerial('FindItem') == UO.GetSerial('MyFriendObject') then
...
Чтобы узнать тип используйте GetGraphic
