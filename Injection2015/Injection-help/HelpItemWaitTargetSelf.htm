Описание: WaitTargetSelf •

Синтакс:
Консоль: _WaitTargetSelf
Из скрипта: void UO.WaitTargetSelf()

Введено с версии: <=1501.17

Тема: Прицелы
Ключевые слова: Command, ObjHandling, Scripting, Targeting, WaitSystem, Void

Устанавливает ловушку прицела на персонаж.
Следующий прицел, который будет запрошен сервером, будет направлен на вашего персонажа.
См. так же WaitTargetType, WaitTargetObject, Cast, UseSkill, BandageSelf.
