Описание: GetY •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetY([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - координату Y (отсчитывается в клетках с запада на восток, запад это слева-вверху).
Если обьект в контейнере, то координату Y в пикселах внутри контейнера. Обычно начало видимого контейнера больше 0 в виду рисунка стенок контейнера.
