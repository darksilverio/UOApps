Описание: SetJournalLine •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: void UO.SetJournalLine(index, text)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Journal, Scripting, Void

Перезаписывает текст в строке журнала номер number новым текстом text. Получить текст можно командой Journal.
