Описание: Count •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.Count(type, [color, container])

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, TypeHandling, Utility, Function

Возвращает суммарное количество во всех стопках обьекта указаного типа (и цвета если указано) в вашем бекпаке и подсумках. Учитываются только те подсумки которые открывались, иначе их содержимое не известно.
