Описание: ContainerOf •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.ContainerOf(serial)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - контейнер, в котором находится. 0xFFFFFFFF означает неизвестно или на земле.
String uo.ContainerOf([String/Number] serial)

Вызов из скрипта:
Code:

sub EatPig(pig) #Кушает свинью
   if uo.ContainerOf(pig) == uo.GetSerial(backpack) then #Сравниваем контейнер со свинкой с нашим паком
      uo.UseObject(pig) #Если лежит в паке - используем ее
   else #Свинья чужая((
      uo.Print('This pig is not your!')
   endif
end sub
