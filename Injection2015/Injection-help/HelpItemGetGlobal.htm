Описание: GetGlobal •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetGlobal(name)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Scripting, System, Function

Возвращает значение глобальной текстовой переменной, связанной с идентификатором name.

Значения таких переменных глобальные для всех скриптов текущей копии Injection.

См. так же GetEasyUO.
