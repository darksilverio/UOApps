Описание: CanChangeName •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.CanChangeName(serial)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает указатель на возможность изменения имени указанного персонажа.
Актуально применять для скриптов, которые изменяют имя живности. Для корректной работы необходимо использовать в связке с uo.GetStatus(serial)
Number uo.CanChangeName([String/Number] serial)

Вызов из скрипта:
Code:

sub ChangePetName(serial, newname) #Изменить имя указанному зверю (персонажу)
   uo.GetStatus(serial) #Запрос статуса (в этом пакете приходит флаг возможности изменения имени)
   wait(100) #Задержка на получение пакета от сервера
   uo.Rename(serial, neewname) #Вызываем функцию для смены имени
end sub
