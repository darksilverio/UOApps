Описание: SkillVal •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.SkillVal(skill, variant)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: CastSkill, Scripting, Function

Возвращает величину умения в десятых процента, т.е. 12.3%=123.
Если задано значение параметра case то можно получить такие значения:

  • case=0: величина умения в десятых процента
  • case=1: величина умения в десятых процента (real)
  • case=2: увеличение умения с момента загрузки или сброса в десятых процента
  • case=3: статус умения (открыт, закрыт, уменьшается)
