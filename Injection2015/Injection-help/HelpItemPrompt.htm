Описание: Prompt •



Тема: Символы
Ключевые слова: CraftMenu, Symbol, Void

На многих шардах используется для создания вещей крафтом, инскриптом, при применении некоторой магии используется меню как на рисунке, выводящееся сверху слева. Подсказкой Prompt называют заголовок меню (сверху), в котором шард обычно пишет фразу типа "What do you wish to craft?" а выбором Choice называют надпись снизу появляющуюся при наведении на один из предлагаемых разделов. Надпись может быть использована в автообработчиках меню крафта AutoMenu и WaitMenu. Не обязательно использовать его полностью, достаточно части по которой можно узнать меню среди других.
