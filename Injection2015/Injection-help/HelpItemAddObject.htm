Описание: AddObject •

Синтакс:
Консоль: _AddObject name [serial]
Из скрипта: void UO.AddObject(name, [serial])

Введено с версии: <=1501.17

Тема: Системные
Ключевые слова: Command, ObjHandling, Scripting, System, Void

Список обьектов находится в закладке Objects. Добавить новый обьект можно как там, так и командой AddObject. Для этого вам надо указать имя обьекта name. Если вы сразу не задали обьект Object который хотите добавить, то Injection выдаст подсказку "What is object?" и предложит выбрать обьект прицелом.
