Описание: SetSeason •

Синтакс:
Консоль: _SetSeason index
Из скрипта: void UO.SetSeason(index)

Введено с версии: <=1501.17

Тема: Приложения
Ключевые слова: Command, Scripting, System, Utility, Void

Устанавливает сезон согласно season_num:

  • 0 - по умолчанию (текущий)
  • 1 - spring (весна)
  • 2 - summer (лето)
  • 3 - fall (осень)
  • 4 - winter (зима)
  • 5 - desolation (засуха)
