Описание: SetCatchBag •

Синтакс:
Консоль: _SetCatchBag [serial]
Из скрипта: void UO.SetCatchBag([serial])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, System, Void

Активирует режим в котором все упавшие в бекпак обьекты будут перекладываться в указаный контейнер. Не забывайте если решили использовать эту команду, отключить её командой UnsetCatchBag.
