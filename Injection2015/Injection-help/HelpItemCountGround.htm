Описание: CountGround •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.CountGround(type, [color])

Введено с версии: <=1501.17

Доступно только из модуля Script.dll

Тема: Команды скриптов
Ключевые слова: Scripting, TypeHandling, Utility, Function

Возвращает суммарное количество во всех стопках обьекта указаного типа (и цвета если указано) на земле возле персонажа.
См. так же count, FindType
