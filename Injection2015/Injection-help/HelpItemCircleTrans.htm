Описание: CircleTrans •

Синтакс:
Консоль: _CircleTrans [state]
Из скрипта: Function UO.CircleTrans([state])

Введено с версии: <=1501.17

Тема: Приложения
Ключевые слова: Command, Scripting, Utility, Function

Изменяет состояние галочки в опциях клиента Transparency Circle.
При вызове из скрипта:

  • Number uo.CircleTrans() - возвращает текущее состояние
  • void uo.CircleTrans(Number state) задает значение для переключения

Вызов из скрипта:
Code:

sub ChangeTrans()
   uo.CircleTrans(not uo.CircleTrans()) #Переключит состояние
end sub


Вызов из хоткеев:
Code:

circletrans - переключает состояние галочки
circletrans value - устанавливает состояние галочки в value (0 - выключена; иное значение - включена)
