Описание: WaitTargetObject •

Синтакс:
Консоль: _WaitTargetObject serial [serial2]
Из скрипта: void UO.WaitTargetObject(serial, [serial2])

Введено с версии: <=1501.17

Тема: Прицелы
Ключевые слова: Command, ObjHandling, Scripting, Targeting, WaitSystem, Void

Устанавливает на обьект ловушку прицела.
Следующий прицел, который будет запрошен сервером, будет направлен на этот обьект.
Если задан обьект 2, то после срабатывания ловушки будет установлена ловушка на обьект 2.
См. так же WaitTargetType, WaitTargetGround, Poison, WaitTargetTile, FindType.
