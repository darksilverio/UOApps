Описание: SetArm •

Синтакс:
Консоль: _SetArm name
Из скрипта: void UO.SetArm(name)

Введено с версии: <=1501.17

Тема: Одеяние
Ключевые слова: Command, Equipment, Scripting, System, Void

Запоминает одетое на персонаже вооружение под именем набора Name. Одеть его позже можно командой Arm. Убрать набор из памяти можно командой Unsetarm.

Пример использования команд arm и disarm

Пример ждёт безоружным пока не пнут, после чего одевает то оружие что было надето при запуске скрипта.

Code:
sub test()
UO.SetArm('default')
UO.Disarm()
while UO.GetHP() == UO.GetMaxHP()
   wait(50)
wend
UO.Arm('default')
end sub
