Описание: SetDress •

Синтакс:
Консоль: _SetDress name
Из скрипта: void UO.SetDress(name)

Введено с версии: <=1501.17

Тема: Одеяние
Ключевые слова: Command, Equipment, Scripting, System, Void

Запоминает одетую на персонаже одежду под именем набора Name. Одеть его позже можно командой Dress. Убрать набор из памяти можно командой Unsetdress.
