Описание: GetQuantity •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetQuantity(serial)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - количество в стопке. Чтобы узнать суммарное количество вещей указаного типа используйте Count. Чтобы узнать количество стопок FindType.
