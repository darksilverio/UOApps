Описание: FontColor •

Синтакс:
Консоль: _FontColor color
Из скрипта: void UO.FontColor(color)

Введено с версии: <=1501.17

Тема: Системные
Ключевые слова: Command, Scripting, System, Utility, Void

Устанавливает цвет речи вашего персонажа. В виде FontColor dye команда выводит меню цвета для выбора в нём. Цвет совпадает только иногда и приблизительно в таком случае.
