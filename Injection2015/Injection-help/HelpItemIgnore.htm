Описание: Ignore •

Синтакс:
Консоль: _Ignore serial [state]
Из скрипта: void UO.Ignore(serial, [state])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, System, Void

Указывает что обьект должен игнорироваться (или не игнорироваться, с параметром off) при последующих командах поиска, например FindType. Разрешить поиск всех обьектов снова можно командой IgnoreReset
