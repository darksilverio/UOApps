Описание: IsNPC •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.IsNPC([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - живой ли обьект (вы сами, другие игроки, NPC, монстры). Обычно у таких обьектов можно узнать злобность.
