Описание: Poison •

Синтакс:
Консоль: _Poison serial
Из скрипта: void UO.Poison(serial)

Введено с версии: <=1501.17

Тема: Приложения
Ключевые слова: Command, ObjHandling, Scripting, Utility, Void

Применяет Skill Poisoning к бутылке яда и обьекту. Если вместо обьекта задан параметр it то обьектом будет являться сама бутылка с ядом. Порядок целей на вашем шарде (сперва бутылка потом обьект или наоборот) выставляется в закладке Main, опция Poison: bottle then obj.

Для работы команды необходим прописаный тип обьекта {type} с именем poison (бутылки с ядом)
