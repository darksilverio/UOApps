Описание: Set •

Синтакс:
Консоль: _Set variable value
Из скрипта: void UO.Set(variable, value)

Введено с версии: <=1501.17

Тема: Системные
Ключевые слова: Command, Scripting, System, Utility, Void

В команде изменяются множество настроек Injection. Посмотреть какие настройки знает ваша версия вы можете набрав Set без параметров. С одним параметром (variable) команда выдаёт текущее значениеvariable. С двумя параметрами (variable,value) значение variable устанавливается в value.
Список variable:

  • distance - дистанция использования предметов
  • lasttarget - последний (Object,обьект) на который был наведён прицел
  • prefix - командный префикс (по умолчанию запятая ,)
  • bankhack - автоприменение BoxHack к банковому контейнеру
  • finddistance - дистанция поиска на земле
  • gumpunicode - переключает использование unicode в гампах
  • maxsell - максимальное количество вещей на продажу за один раз
  • buydelay - задержка перед покупкой, мс
  • injectecho - добавляет сообщения Injection в журнал
  • season - см. SetSeason
  • dressspeed - см. SetDressSpeed
  • grabdelay - задержка в мс при перемещении предметов используя комманды Injection
  • statlinelen - длина линий здоровья/маны/стамины в заголовке окна
  • concolor - цвет сообщений Injection
  • quiet - отключение всех сообщений Injection
  • gdislower - включение специального замедлителя, мс, понижающего использование CPU клиентом
  • soundecho - отображение звуков в журнале
  • animecho - отображение анимации в журнале
  • norbcheck - отключение проверки идентичности гампа в командах рунбуков (Рунбуки)
  • norbcalc - отключение вычислений ID кноки на основе номера в командах рунбуков (Рунбуки). В включеном состоянии просто полагается ID=номер.
  • nogump - отключение обработки гампов. Если у вас при появлении большого гампа постоянно чтото слетает попробуйте єту опцию.
  • blocktargets - блокирование системы прицелов клиента. полезно при использовании в макросах команды Target
  • blockstop - остановка движения при первом запрете от сервера
  • transmain - устанавливает прозрачность окна Injection
  • topmain - устанавливает окну Injection свойство "поверх окон"
  • statsdisplaypercent - отображать в заголовке характеристики персонажа в % соотношении или в реальных значениях
  • staminapatch - изменение состояния галочки Stamina Patch
  • fpspatch - изменение состояния галочки FPS Patch
  • corpsesautoopen - изменение состояния галочки Corpses Autoopen
  • undead - изменение состояния галочки Undead
  • passhotkeys - изменение состояния галочки Pass Hotkeys
  • trackworlitems - изменение состояния галочки Track Worl Items
  • verboselogging - изменение состояния галочки Verbose Logging
  • verbosemenu - изменение состояния галочки Verbose Menu
  • destroyservermenus - изменение состояния галочки Destroy Server Menus
  • stealthcounter - изменение состояния галочки Stealth Counter
  • smoothwalk - изменение состояния галочки Smooth Walk
  • permitvarsloopback - изменение состояния галочки Permit Vars Loopback
  • gamewindowsize - изменение состояния галочки Game Window Size
  • autostart - изменение состояния галочки Autostart
  • statusbarsfix - изменение состояния галочки Statusbars Fix
  • inversebuylist - изменение состояния галочки Inverse Buylist
  • gamewindowsizex - указать ширину игрового окна
  • gamewindowsizey - указать высоту игрового окна
  • pfnomap - изменение алгоритма вычисления поиска пути
  • sbgreen - отображение статусбара в цветном заголовке УО в стиле ассиста
  • corpsetimer - установить время в мс для ожидания открытия трупов при использовании функции Corpses Autoopen, но несоблюдении ее требований (труп слишком далеко)
  • autoparty - автоматические действия с пати: 1 - принять; 2 - отклонить; иное значение - на выбор пользователя
  • displaysepar - вкл/выкл отображение разделителей (белых полосок) в цветном заголовке
  • displayuo - вкл/выкл отображение значка УО в цветном заголовке
