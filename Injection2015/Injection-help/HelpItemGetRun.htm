Описание: GetRun •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetRun([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - бежит ли он. Похоже что кроме как о игроке правильная информация не приходит.
