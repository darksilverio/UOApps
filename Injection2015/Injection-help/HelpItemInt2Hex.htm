Описание: Int2Hex •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.Int2Hex(number)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Scripting, System, Utility, Function

Преобразует число из десятичного формата в шестнадцатеричный "0x12345678". Может использоваться для сравнения принадлежности чего-то диапазону.

Code:
for i = ClumsyScroll to RecallScroll - 1
if UO.Count(UO.Int2Hex(i)) then
...
Для обратного преобразования используется Hex2Int
