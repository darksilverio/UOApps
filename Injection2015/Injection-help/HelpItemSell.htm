Описание: Sell •

Синтакс:
Консоль: _Sell list [name]
Из скрипта: void UO.Sell(list, [name])

Введено с версии: <=1501.17

Тема: Покупка и продажа
Ключевые слова: BuySell, Command, Scripting, Void

К имени вендора vendorname дописывает ключевое слово продажи из файла injection.xml (обычно это sell) и произносит эту фразу, после чего продаёт в меню продажи, которое покажет вендор, вещи согласно списку listname.

Посмотреть, отредактировать и создать новый список можно нажав кнопку Buy/sell закладки Main или воспользовавшись командой ,shop

В случае необходимости можно включить задержку перед продажей командой Set buydelay

Чтобы изменить фразу buy для своего сервера найдите в injection.xml:

Code:
server name="My Server Name"
            ...
            sell="sell"
            ....

и замените "sell" на подходящее для вашего шарда.
