Описание: GetZ •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetZ([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - координату Z (в точках). Вода в основном имеет высоту -5, земля в ровной местности обычно около 0. Высота на которой находится персонаж известна не точно; она обновляется после захода на сервер, после рекола и, если сервер поддерживает, команды Resend.
Если обьект в контейнере, то координата Z обычно 0.
