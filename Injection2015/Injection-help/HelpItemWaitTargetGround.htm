Описание: WaitTargetGround •

Синтакс:
Консоль: _WaitTargetGround type [color]
Из скрипта: void UO.WaitTargetGround(type, [color])

Введено с версии: <=1501.17

Тема: Прицелы
Ключевые слова: Command, Scripting, Targeting, TileHandling, WaitSystem, Void

Ищет на земле обьект по типу (и цвету если задано) после чего устанавливает на этот обьект ловушку прицела.
Следующий прицел, который будет запрошен сервером, будет направлен на этот обьект.
См. так же WaitTargetType, WaitTargetObject, FindType.
