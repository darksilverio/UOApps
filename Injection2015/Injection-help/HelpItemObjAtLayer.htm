Описание: ObjAtLayer •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.ObjAtLayer(layer)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Equipment, ObjHandling, Scripting, Function

Возвращает обьект находящийся на указаном слое Layer, например в правой руке (Layer = "Rhand"). Со слоями работают так же команды Equip, EquipT, Unequip.
