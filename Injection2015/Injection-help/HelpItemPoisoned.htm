Описание: Poisoned •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.Poisoned([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - отравлен ли он. Имеются в виду живые персонажи, эта команда не заменяет Skill Taste Identification.
