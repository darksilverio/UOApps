Описание: GetX •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetX([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - координату X (отсчитывается в клетках с севера на юг, север это справа-вверху).
Если обьект в контейнере, то координату X в пикселах внутри контейнера. Обычно начало видимого контейнера больше 0 в виду рисунка стенок контейнера.
