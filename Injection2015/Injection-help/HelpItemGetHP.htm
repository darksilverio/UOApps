Описание: GetHP •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetHP([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - здоровье персонажа. Для другого персонажа кроме игрока обычно не имеет смысла абсолютное значение здоровья, так как выражается в абстрактных единицах, важно только отношение к максимальному здоровью GetMaxHP.

Пример использования команд arm и disarm

Пример ждёт безоружным пока не пнут, после чего одевает то оружие что было надето при запуске скрипта.

Code:

sub test()
uo.SetArm("default")
uo.Disarm()
while uo.GetHP()==uo.GetMaxHP()
   wait(50)
wend
uo.Arm("default")
end sub
