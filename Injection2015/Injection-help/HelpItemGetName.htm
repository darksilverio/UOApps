Описание: GetName •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: Function UO.GetName([serial])

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: ObjHandling, Scripting, Function

Возвращает параметр обьекта (по умолчанию - персонажа игрока) - имя обьекта. Если сервер никогда не выводил его на экран (как это делается например после одиночного щелчка Click на обьекте или использования All Names в клиенте), то имя неизвестно.
