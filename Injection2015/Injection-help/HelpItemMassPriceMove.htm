Описание: MassPriceMove •

Синтакс:
Консоль: _MassPriceMove delay [maxitems price color]
Из скрипта: void UO.MassPriceMove(delay, [maxitems, price, color])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, ObjHandling, Scripting, Utility, Void

Для шардов в которых цена предмета задаётся в момент когда предмет ложится в сумку вендору (например AoP, Wolfpack), можно использовать команду MassPriceMove для задания цены большому количеству одинаковых предметов.

Команда использует для своей работы основу команды MassMove, с дополнительным параметром price - цена единицы товара.

Для использования сперва введите команду MassPriceMove с параметрами, потом переместите один эталонный предмет вручную в сумку или на сумку вендора.
