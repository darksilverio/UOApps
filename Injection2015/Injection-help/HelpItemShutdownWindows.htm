Описание: ShutdownWindows •

Синтакс:
Консоль: _ShutdownWindows state
Из скрипта: void UO.ShutdownWindows(state)

Введено с версии: <=1501.17

Тема: Системные
Ключевые слова: Command, Scripting, System, Utility, Void

Завершает работу MS Windows. normal - обычное завершение, forced - с закртием не отвечающих программ.
Под ОС WinXP скорее всего не сработает, вследствие недостаточных прав доступа. В таком случае может помочь команда Launch shutdown -s
