Описание: Undress •

Синтакс:
Консоль: _Undress
Из скрипта: void UO.Undress()

Введено с версии: <=1501.17

Тема: Одеяние
Ключевые слова: Command, Equipment, Scripting, Void

Снимает одетую одежду. Рекомендуется выставить задержку между одеваниями отдельных вещей командой SetDressSpeed.

Если обьявлен обьект с именем undressbag то он будет использован как контейнер для раздевания.
