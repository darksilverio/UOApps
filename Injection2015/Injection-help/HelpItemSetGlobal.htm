Описание: SetGlobal •

Синтакс:
Консоль: Только из скрипта.
Из скрипта: void UO.SetGlobal(name, value)

Введено с версии: <=1501.17

Тема: Команды скриптов
Ключевые слова: Scripting, System, Utility, Void

Устанавливает значение глобальной текстовой переменной, связанной с идентификатором Name.

Значения таких переменных глобальные для всех скриптов текущей копии Injection.

См. так же SetEasyUO.
