Описание: UseFromGround •

Синтакс:
Консоль: _UseFromGround type [color]
Из скрипта: void UO.UseFromGround(type, [color])

Введено с версии: <=1501.17

Тема: Работа с объектами
Ключевые слова: Command, Scripting, TypeHandling, Void

Ищет обьект на земле по типу (и цвету если указано) и использует его (двойным щелчком). Смотрите так же команды UseType, CountGround.
