Описание: Disarm •

Синтакс:
Консоль: _Disarm
Из скрипта: void UO.Disarm()

Введено с версии: <=1501.17

Тема: Одеяние
Ключевые слова: Command, Equipment, Scripting, Utility, Void

Разоружает персонаж. Рекомендуется выставить задержку между снятием вещей командой SetDressSpeed.

Если обьявлен обьект с именем disarmbag то он будет использован как контейнер для разоружения.

Пример использования команд arm и disarm

Пример ждёт безоружным пока не пнут, после чего одевает то оружие что было надето при запуске скрипта.

Code:

sub test()
uo.SetArm("default")
uo.Disarm()
while uo.GetHP()==uo.GetMaxHP()
 wait(50)
wend
uo.Arm("default")
end sub
