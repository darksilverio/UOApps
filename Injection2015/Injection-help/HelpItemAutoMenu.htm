Описание: AutoMenu •

Синтакс:
Консоль: _AutoMenu prompt choice
Из скрипта: void UO.AutoMenu(prompt, choice)

Введено с версии: <=1501.17

Тема: Меню крафта
Ключевые слова: Command, CraftMenu, Scripting, WaitSystem, Void

Команды WaitMenu и AutoMenu используются для автоматического выбора в специальном меню крафта, согласно предоставленых параметров Prompt и Choice.

Каждое использование команды AutoMenu добавляет в специальный список очередной выбор в меню, после чего он остаётся активным всё время, до задания команды CancelMenu. Список текущих пар Prompt/Choice выводится на экран. В случае если сервер пошлёт подходящее к Prompt меню, Injection сделает выбор Choice и будет продолжать ожидать список выборов. Вы не можете применить эту команду если вам требуется сделать разные выборы в меню с одинаковыми названиями Prompt, в таком случае используется команда WaitMenu
