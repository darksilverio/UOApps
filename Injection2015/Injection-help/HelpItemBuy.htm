Описание: Buy •

Синтакс:
Консоль: _Buy list [name]
Из скрипта: void UO.Buy(list, [name])

Введено с версии: <=1501.17

Тема: Покупка и продажа
Ключевые слова: BuySell, Command, Scripting, WaitSystem, Void

К имени вендора name дописывает ключевое слово покупки из файла injection.ini (обычно это buy) и произносит эту фразу, после чего покупает в меню покупки, которое покажет вендор, вещи согласно списку list.

Посмотреть, отредактировать и создать новый список можно нажав кнопку Buy/Sell закладки Main или воспользовавшись командой Shop

В случае необходимости можно включить задержку перед покупкой командой Set BuyDelay
